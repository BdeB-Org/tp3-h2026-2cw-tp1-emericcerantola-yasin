Clinique Vétérinaire

Application Web dynamique connectée à une API Oracle ORDS

Accueil Propriétaires Animaux Vétérinaires Consultations

Bienvenue

Ce projet permet de gérer les dossiers médicaux d'une clinique vétérinaire. Il consomme une API REST Oracle ORDS à partir du navigateur avec fetch().

Fonctionnalités incluses

  • Consultation et gestion des propriétaires
  • Consultation des animaux
  • Consultation et gestion des vétérinaires
  • Consultation des consultations et leurs traitements

Avant de démarrer

  1. Créer la base Oracle avec le script SQL fourni
  2. Activer ORDS sur le schéma et les tables
  3. Vérifier les endpoints dans le navigateur
  4. Ouvrir le dossier dans VS Code
  5. Lancer avec Live Server
Important : l'URL de base ORDS se modifie dans js/api.js à la constante BASE_URL.

Pages du projet

  • proprietaires.html : liste, ajout, suppression
  • animaux.html : liste et recherche
  • veterinaires.html : liste, ajout, suppression
  • consultations.html : liste des consultations et traitements associés

Architecture

Frontend : HTML, CSS, JavaScript

API : Oracle ORDS

Base de données : schéma clinique

Exécution : VS Code + Live Server
